Accessing Elements

Using the id ( #header )

Access this by using getElementById()

Using the class ( .header )

Access this by using getElementByClassName()

Using the Tag ( paragraph tag )

1. Access this by using getElementByTagName()

2. Access this by using getElementByTagName()

Using the Query selector ( with ID of 'query' )

Access this by using querySelector()

Using the Query selector All ( with ID of 'query-all' )

1. Access this by using querySelectorAll()

2. Access this by using querySelectorAll()
